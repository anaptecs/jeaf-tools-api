# JEAF Tools API README #

This repository contains the source code of JEAF Tools API. JEAF Tools is a collection of tools classes that can be used to build enterprise applications. 

JEAF Tools provides functionality for collections, strings, encryption, files, reflection, performance monitoring and many more. 

## Links ##
For further information please refer to:

* [JEAF Tools Getting started](https://anaptecs.atlassian.net/l/c/xQwBqiwH)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.tools/jeaf-tools-api)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.tools/jeaf-tools-api)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
